## Navigation tips

**General**

* ++f++ , ++s++ , ++slash++ : open search dialog

**Search**

* ++arrow-down++ , ++arrow-up++ : select next / previous result
* ++enter++ : follow selected result
* ++esc++ , ++tab++ : close search dialog
